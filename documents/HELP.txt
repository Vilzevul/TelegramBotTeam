<b>Инструкция для пользователей Telegram-бота</b>
<b><i>Для начала работы с ботом наберите команду /start</i></b>
Бот с Вами поздоровается и предложит меню из кнопок:
1.	Приют для собак.
<i>При выборе кнопки «Приют для собак» бот откроет список, где можно узнать,
   где находится приют, правила приюта, как можно взять собаку и прислать отчет о взятой собаке из приюта.</i>
2.	Приют для кошек.
<i>При выборе кнопки «Приют для кошек» бот откроет список, где можно узнать,
   где находится приют, правила приюта, как можно взять кошку и прислать отчет о взятой кошке из приюта.</i>
3.	Оставить сообщение.
4.	Помощь.
<i>Инструкция для пользователей Telegram-бота.</i>
5.	Оставить контакты.
<i>В появившемся всплывающем окне необходимо дать согласие на обработку Вашего телефонного номера чат-ботом (нажмите Share).</i>
